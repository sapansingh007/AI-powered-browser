#  AI Browser & Summarizer

A sophisticated multi-tab browser with integrated AI-powered content summarization and translation capabilities, built with Flutter and Riverpod.

##  Project Overview

**AI Browser & Summarizer** is a Chrome-like browser that combines web browsing with intelligent content analysis. It features multi-tab management, offline mode, file downloads, and AI-powered summarization with translation support.

###  Key Features
-  iOS (WebKit-based)
-  Web (PWA-ready)
